# Configure Basic Auth in Zuplo

## Background
You have a Zuplo API project at `/home/user/myproject`. You need to protect the `GET /todos` route with Basic Authentication.

## Requirements
- Add a `basic-auth-inbound` policy named `basic-auth-inbound` to `config/policies.json`.
- Configure it with one account: username `admin`, password `secretpassword`, and some data `{"role": "admin"}`.
- Apply the `basic-auth-inbound` policy to the `GET /todos` route in `config/routes.oas.json`.

## Constraints
- Project path: /home/user/myproject
- The project is a standard Zuplo project generated by `create-zuplo-api`.
